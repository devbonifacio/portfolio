## 🚀 Sobre o Projeto

Portfólio pessoal criado para apresentar minhas habilidades, projetos e formas de contato. O site conta com suporte a dois idiomas (🇧🇷 Português / 🇺🇸 Inglês) e um design moderno focado em desenvolvedores.

## ✨ Funcionalidades

- 🌐 Troca de idioma PT / EN
- 🎨 Design dark moderno com animações
- 📱 Totalmente responsivo
- ⚡ Scroll reveal nas seções
- 🟢 Indicador de disponibilidade

## 🛠️ Tecnologias

- HTML5
- CSS3 (animações, variáveis CSS, grid, flexbox)
- JavaScript puro (sem frameworks)
