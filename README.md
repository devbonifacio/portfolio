## 🚀 Sobre

Portfólio pessoal criado para apresentar meus projetos, habilidades e caminho como desenvolvedor. Inclui:

- 🏥 **Experiência profissional** — Estágio no sistema hospitalar HiSi (Santa Casa de Vila Verde)
- 💻 **Projetos full-stack** — DevMatch, Finance-Flow, TaskFlow e mais
- 🎨 **Design premium** — Dark mode, animações suaves e foco em UX
- 🌍 **Bilíngue** — PT / EN com persistência da preferência

---

## ✨ Funcionalidades

- 🌐 Troca de idioma PT / EN (com `localStorage`)
- 🎨 Dark theme moderno com efeitos de glow e grid de fundo
- 📱 Totalmente responsivo (mobile-first)
- 📍 Timeline de experiência profissional
- 🔍 Filtro de projetos por categoria (Full Stack / Frontend / Profissional)
- ⚡ Scroll reveal com `IntersectionObserver`
- 🟢 Indicador de disponibilidade
- ♿ Suporte a `prefers-reduced-motion` para acessibilidade

---

## 🛠️ Tech Stack

- **HTML5** — semântico, acessível
- **CSS3** — custom properties, grid, flexbox, animações
- **JavaScript ES6+** — modular, sem dependências
